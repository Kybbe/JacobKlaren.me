Eddler

Eddler 10 week internship
10 W / November 2019 - January 2020

Eddler is a smaller swedish techcompany which formerly was called "MatematikVideo" in Swedish or "MathematicsVideo" in english ofcourse.
Eddler specializes in math learning online through video or thorough walkthroughs.
They also have several other courses like Physics and Programming which work in the same way.

My job at Eddler was really to learn how a small company works from the ground up.
I also got a small task to integrate a Mathgame, which should be easy, modular and engaging.
The game in its final form was made so anyone can update it, used the same artsyle as Eddlers website,
and was very easy to change individual levels on etc.
Eddler was to use the mathgame in some Premium courses, but it was also put as a seperate game so it could be found easily on Google.
When Eddler wants to use the game in their courses, it has a custom tag, which takes in all level variables either from file or in plain text, making it very user fiendly.
It also supports an "admin-site" where Eddler or admins could see how individual users fairs in each game (making it so teachers can see if their students do their homework for example, or what they need extra help with in the future)
